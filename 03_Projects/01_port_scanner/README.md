# TCP Port Scanner Project

This project documents my learning process while building a network port scanner in Python.

Instead of building everything at once, I improved the scanner step by step to understand how network scanning works.

---

## Versions

```
v1_basic
```

Single-threaded TCP scanner.

Features:

```
- Common ports and custom range
- Timeout handling
- Basic output
```

This version helped me understand how TCP connections work using Python sockets.

---

```
v2_threaded
```

Improved version of the scanner with better performance.

Features:

```
- Multi-threaded scanning
- Faster performance
- Service detection
- Banner grabbing
```

This version focuses on learning threading and improving scanning speed.
